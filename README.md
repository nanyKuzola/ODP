# Oncology Data Platform

Mini plataforma de análise de dados oncológicos baseada em dados públicos do OpenDataSUS.

Arquitetura:

OpenDataSUS dataset  
↓  
ETL pipeline (Python)  
↓  
Data Lake (raw / processed)  
↓  
API Django  
↓  
Dashboard analítico  

Objetivo: simular a arquitetura de integração de dados descrita no projeto de monitoramento oncológico.
